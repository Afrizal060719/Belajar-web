Belajar operator kesamaan ==, kesamaan ketat ===, and &&

Operator kesamaan : == berfungsi untuk membandingkan nilai

Operator kesamaan ketat : === berfungsi untuk membandingkan nilai dan tipe data jadi ini lebih aman

Operator and : && berfungsi untuk syrat melanjutkan kondisi, operator ini berjalan dari kiri ke kanan dan jika 2 nilai salah maka hasilnya false dan jika 2 nilai benar maka true / melanjutkan pengkondisian
